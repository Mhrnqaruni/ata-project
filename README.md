# ata-project

## About This Project

ATA (Academic Teaching Assistant) is a comprehensive educational platform designed to streamline classroom management and enhance student engagement through interactive assessments and real-time quizzing. Built with a modern tech stack featuring FastAPI (Python) on the backend and React on the frontend, the platform enables teachers to create and manage classes, conduct live quiz sessions with automatic grading, track student performance through detailed analytics, and facilitate seamless student-teacher interactions.
